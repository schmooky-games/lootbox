## Получение информации о конкретном лутбоксе

**Endpoint:** `GET /lootboxes/{lootbox_id}`

Возвращает информацию о конкретном лутбоксе по его идентификатору.

**Параметры пути:**
- `lootbox_id` (строка): Идентификатор лутбокса.

**Ответ:**
Возвращает объект лутбокса (`EqualLootbox`, `WeightedLootbox` или `ExclusiveLootbox`).

## Обработка ошибок

API может возвращать следующие ошибки:

- `400 Bad Request`:
  - `1001`: Лутбокс не найден.

При возникновении ошибки API возвращает соответствующий статус-код и детальное описание проблемы.
